Date Addition Function Test

Testing the new date addition functionality: A+B where A is date and B is number of days

Test Case 1: Delivery Date + Add Days = Due Date

Delivery Date: 10/14/2014

Add Days: 1

Formula: A+B (where A=delivery_date, B=add_days)

Delivery Date: 10/14/2014
