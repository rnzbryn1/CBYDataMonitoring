📊 Excel-Style SUMIFS Function

🎯 Perfect Excel Compatibility

Your SUMIFS function now works exactly like Excel! The first parameter specifies which sheet (template) to search in, just like in Excel.

📋 Excel Syntax
=SUMIFS('RM DELIVERY'!C:C, 'RM DELIVERY'!A:A, "Banana Saba", 'RM DELIVERY'!B:B, "10")
🚀 Our System Syntax
=SUMIFS('RM DELIVERY', 'Accepted QTY', 'Item Description', 'Banana Saba', 'Month #', '10')

📝 How It Works

Your exact example:

=SUMIFS('RM DELIVERY', 'Accepted QTY', 'Item Description', 'Banana Saba', 'Month #', '10')

This formula will:

  1. Find the 'RM DELIVERY' template - your encoding sheet
  2. Look for 'Item Description' column and find all "Banana Saba"
  3. Look for 'Month #' column and find all "10"
  4. Find entries that match BOTH conditions (AND logic)
  5. Sum all values in 'Accepted QTY' column for those matching entries

🔧 Key Features

  • Sheet-Specific: Searches only in the specified template (like Excel sheets)
  • Case-Insensitive: "Banana Saba" = "banana saba" = "BANANA SABA"
  • Multiple Conditions: Supports unlimited condition pairs
  • Smart Caching: Results cached for performance
  • Auto-Refresh: Updates automatically when data changes

💡 More Examples

Single Condition:
=SUMIFS('RM DELIVERY', 'Accepted QTY', 'Item Description', 'Banana Saba')
Multiple Conditions:
=SUMIFS('RM DELIVERY', 'Accepted QTY', 'Item Description', 'Banana Saba', 'Month #', '10', 'Year', '2024')
Different Sheet:
=SUMIFS('SALES ORDER', 'Total Amount', 'Status', 'Completed', 'Region', 'Manila')
