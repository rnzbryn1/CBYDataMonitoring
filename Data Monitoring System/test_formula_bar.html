Formula Bar Feature Test

Test Results:

Manual Test Instructions:

  1. Open the main application (pcd.html)
  2. Select a template and load some data
  3. Right-click on any cell and select "Compute"
  4. Test 1 - Empty Formula: The formula bar should be empty if no formula is applied
  5. Test 2 - Apply Formula: Enter a formula like "=Price * Quantity" and click Apply
  6. Test 3 - Show Formula: Right-click the same cell again and select "Compute" - the formula should now appear in the formula bar
  7. Test 4 - Remove Formula: Clear the formula bar (make it empty) and click Apply/Update
  8. Test 5 - Verify Removal: Right-click the same cell again - the formula bar should be empty again
  9. Test 6 - Column Formula: Test with "Whole Column" mode as well

Expected Features:

  • Formula bar shows current applied formula when opening compute modal
  • Clearing formula and clicking update removes the applied formula
  • Works for both single cell and whole column formulas
  • Formulas are saved/removed from database correctly
