Test Monitoring Computation Auto-Update

Test Description

This test verifies that monitoring templates update immediately when computations are applied.

Expected behavior: When formulas are computed in a monitoring template, the results should appear immediately without requiring a manual refresh.

Test Steps

  1. Open the main application
  2. Navigate to a monitoring template with applied formulas
  3. Make changes to the corresponding encoding template
  4. Observe if the monitoring template updates immediately
